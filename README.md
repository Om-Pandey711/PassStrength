# PassStrength Analyzer

**Measure. Improve. Secure.**

A simple yet effective tool to analyze password strength and help users understand how secure their passwords really are.

---

## About PassStrength Analyzer

**PassStrength Analyzer** is a security-focused utility designed to evaluate the strength of passwords based on multiple factors such as:

- Length  
- Character variety  
- Complexity  
- Common password patterns  

It is ideal for **cybersecurity learners, developers, and everyday users** who want to improve password hygiene.

---


## Features

- **Password Strength Evaluation**  
  Rates passwords as *Weak, Medium, or Strong*

- **Complexity Analysis**  
  Checks for:
  - Uppercase & lowercase letters  
  - Numbers  
  - Special characters  

- **Weak Pattern Detection**  
  Identifies common passwords and predictable patterns  

- **Fast & Lightweight**  
  Instant analysis with minimal system usage  

- **Beginner Friendly**  
  Clean logic, easy-to-read code  

---
